# 项目全局强制规则（始终生效）
1. 所有文件**强制 UTF-8 无 BOM 编码**，禁止 GBK/ANSI 编码
2. 全场景兼容**中文路径/中文文件名**，路径字符串统一用双引号包裹
3. 代码注释率**≥50%（行级统计）**，注释说明设计原因/边界风险，禁止复述代码
4. 零臆想原则：禁止编造 API、函数名、配置项，缺失信息用 `{{YOUR_XXX}}` 占位并提问
5. Vibicoding 增量编码：单次仅修改 1-2 个关联文件，禁止大规模重构存量代码
